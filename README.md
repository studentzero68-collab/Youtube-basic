# YouTube Basic Clone

A simple, responsive HTML, CSS, and JavaScript project that recreates the basic layout and functionality of YouTube's interface.

## Overview

This project demonstrates front-end web development skills by building a simplified YouTube clone. It features a clean, user-friendly interface with a focus on responsive design and basic interactivity.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Video Grid Layout** - Displays video thumbnails in a clean grid format
- **Navigation Bar** - Includes header with search functionality
- **Sidebar Navigation** - Quick access to different categories
- **Interactive Elements** - Basic JavaScript interactivity for enhanced user experience
- **Modern Styling** - Clean and minimalist CSS styling

## Technologies Used

- **HTML** (78.8%) - Semantic markup for page structure
- **CSS** (19.1%) - Styling and responsive layouts
- **JavaScript** (2.1%) - Interactive features and functionality

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies or installations required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/studentzero68-collab/Youtube-basic.git
```

2. Navigate to the project directory:
```bash
cd Youtube-basic
```

3. Open `index.html` in your web browser:
```bash
open index.html
```

Or simply double-click the `index.html` file to open it.

## Usage

- **Browse Videos** - Scroll through the video grid to view available content
- **Search** - Use the search bar to find videos (basic functionality)
- **Navigate Categories** - Click on sidebar options to filter content
- **Responsive View** - Resize your browser or open on different devices to see responsive design in action
